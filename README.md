# ChatGPT 带你玩转太阳系

ChatGPT有很丰富的太阳系知识，她可以带着我们游遍太阳系，给我们有用的建议和帮助。如果你想去一些你从没去过的地方旅行，可以关注这个项目，让我们一起走遍太阳系，看看不同的风景。

太阳系的美好有诗为证：

```
木卫三的鱼儿
土卫六的虾
谷神星的西瓜
冥王星的猹
火星上的闰土笑哈哈
```
